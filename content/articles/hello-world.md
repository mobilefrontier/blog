经常使用的hugo命令：

生成文章：

```
hugo new articles/weekly-1.md
```

预览：

```
hugo server --buildDrafts
```

生成、更新网站：

```
hugo
```

部署到Github Pages：

```
# mobilefrontier
git add --all
git commit -m "comment"
git push

# mobilefrontier/public
git add --all
git commit -m "comment"
git push
```
